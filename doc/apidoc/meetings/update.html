PUT /v1/users/:user_id/meetings/:id
Update meeting information

Errors

Code Description
400 The meeting has validation errors
404 The meeting was not found

Examples

PUT /v1/users/1/meetings/1
{
  "meeting": {
    "place": "Statues of the Founding Fathers"
  }
}
200
{
  "meeting": {
    "id": 1,
    "place": "Statues of the Founding Fathers",
    "date": "2017-05-05",
    "time": "10:00",
    "links": {
      "self": "/v1/users/1/meetings/1"
    }
  }
}
PUT /v1/users/1/meetings/1
{
  "meeting": {
    "place": ""
  }
}
400
{
  "errors": {
    "name": [
      "can't be blank"
    ]
  }
}
PUT /v1/users/1/meetings/-1
404

Param name Description
meeting

Validations:

required

Meeting place

Validations:

  • Must be a String

Meeting date

Validations:

  • Must be a Date

required

Meeting time

Validations:

optional

Meeting maximum of participants
