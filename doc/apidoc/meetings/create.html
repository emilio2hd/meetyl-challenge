POST /v1/users/:user_id/meetings
Create a meeting specifying a place, date, and time

Errors

Code Description
400 The meeting has validation errors

Examples

POST /v1/users/1/meetings
{
  "meeting": {
    "place": "Columbia, The angelic Monument Tower",
    "date": "2017-05-05",
    "time": "10:00"
  }
}
201
{
  "meeting": {
    "id": 1,
    "place": "Columbia, The angelic Monument Tower",
    "date": "2017-05-05",
    "time": "10:00",
    "links": {
      "self": "/v1/users/1/meetings/1"
    }
  }
}
POST /v1/users/1/meetings
{
  "meeting": {
    "place": ""
  }
}
400
{
  "errors": {
    "place": [
      "can't be blank"
    ]
  }
}
