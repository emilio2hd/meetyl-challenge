POST /v1/users/:user_id/meetings/:id/invite
Invite an user to a meeting

How to Specify Invitation Recurrence¶ ↑

You can specify the recurrence through the { invitation: { recurrence: {} }}

Daily¶ ↑

Daily

recurrence: { type: 'daily' }

Every 3 days

recurrence: { type: 'daily', options: { interval: 3 } }

Weekly¶ ↑

Weekly

recurrence: { type: 'weekly' }

Every 2 weeks

recurrence: { type: 'weekly', options: { interval: 2 } }

Weekly on Mondays

recurrence: { type: 'weekly', options: { day: ['monday'] } }

Every 2 weeks on Mondays

recurrence: { type: 'weekly', options: { interval: 2, day: ['monday'] } }

Monthly¶ ↑

Monthly

recurrence: { type: 'monthly' }

Every 2 months

recurrence: { type: 'monthly', options: { interval: 2 } }

Monthly on the 15th day of the month

recurrence: { type: 'monthly', options: { day_of_month: [15] } }

Monthly on the 10th and 20th days of the month

recurrence: { type: 'monthly', options: { day_of_month: [10, 20] } }

Monthly on the 1st Tuesday and last Tuesday

recurrence: { type: 'monthly', options: { day_of_week: { tuesday: [1, -1] } } }

Every 2 months on the 1st Monday and last Tuesday

recurrence: { type: 'monthly', options: { interval: 2, day_of_week: { monday: [1], tuesday: [-1] } } }

Yearly¶ ↑

Yearly

recurrence: { type: 'yearly' }

Every 2 years

recurrence: { type: 'yearly', options: { interval: 2 } }

Yearly on the 100th days from the beginning and end of the year

recurrence: { type: 'yearly', options: { day_of_year: [100, -100] } }

Yearly in March

recurrence: { type: 'yearly', options: { month_of_year: [:march] } }

Yearly in January and February

recurrence: { type: 'yearly', options: { month_of_year: [:january, :february] } }

Errors

Code Description
400 The invitation has validation errors
404 The meeting was not found

Metadata

:recurrence:
  :type: daily, weekly, monthly, yearly
  :options:
    :day: "[monday, tuesday, wednesday, thursday, friday, saturday, sunday]"
    :day_of_week: "[monday, tuesday, wednesday, thursday, friday, saturday, sunday]"
    :month_of_year: "[january, february, march, april, may, june, july, august, september,
      october, november, december]"

Examples

POST /v1/users/1/meetings/1/invite
{
  "invitation": {
    "invitee_id": 2,
    "recurrence": {
      "type": "weekly",
      "options": {
        "interval": 2,
        "day_of_week": {
          "monday": [
            1
          ],
          "tuesday": [
            -1
          ]
        }
      }
    }
  }
}
201
{
  "invitation": {
    "invitee": "Elizabeth",
    "meeting_link": "http://test.com/v1/meetings/1/NCQ3JDJkMjUxNjhjLTMwOTQtNDc2YS05ZWIzLTIwODUxN2M5YTg3MA=="
  }
}
POST /v1/users/1/meetings/1/invite
{
  "invitation": {
    "recurrence": {
      "type": "badly"
    }
  }
}
400
{
  "errors": {
    "recurrence": [
      "The type 'badly' is not supported"
    ]
  }
}
