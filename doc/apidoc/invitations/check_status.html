GET /v1/users/:user_id/invitations/:id/status
Check the status of an invitation they received

Errors

Code Description
404 The invitation was not found

Metadata

:status:
- pending
- accepted
- declined

Examples

GET /v1/users/1/invitations/1/status
200
{
  "invitation": {
    "status": "accepted"
  }
}
GET /v1/users/1/invitations/-1/status
404
