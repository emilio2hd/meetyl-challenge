PUT /v1/users/:user_id/invitations/:id/execute
Take action on the invitation

Errors

Code Description
400 The invitation has validation errors
404 The invitation was not found

Examples

PUT /v1/users/1/invitations/1/execute
{
  "invitation": {
    "action": "accept"
  }
}
200
{
  "invitation": {
    "status": "accepted"
  }
}
PUT /v1/users/1/invitations/1/execute
{
  "invitation": {
    "action": ""
  }
}
400
{
  "errors": {
    "action": [
      "can't be blank"
    ]
  }
}
PUT /v1/users/1/invitations/-1/execute
404

Params

Param name Description
invitation
required

Invitation info

Validations:

  • Must be a Hash
invitation[action]
required

Action to be executed

Validations:

  • Must be a String


Metadata: 
- accept
- decline
