>>>

POST /v1/users
Add a new user

Errors

Code Description
400 The user has validation errors

Examples

POST /v1/users
{
  "user": {
    "name": "Booker Dewitt"
  }
}
201
{
  "user": {
    "id": 1,
    "name": "Booker Dewitt",
    "links": {
      "self": "/v1/users/1"
    }
  }
}
POST /v1/users
{
  "user": {
    "name": ""
  }
}
400
{
  "errors": {
    "name": [
      "can't be blank"
    ]
  }
}

Params

Param name Description
user
required

User info

Validations:

  • Must be a Hash
user[name]
required

User's name

Validations:

  • Must be a String
