Invitation recurrences

Resource Description
GET /v1/users/:user_id/invitation_recurrences List all user's recurrence, as creator

Invitations

Resource Description
GET /v1/users/:user_id/invitations List user's invitation as invitee
GET /v1/users/:user_id/invitations/:id Retrieve information about a meeting or invitation if they have an invitation
GET /v1/users/:user_id/invitations/:id/status Check the status of an invitation they received
PUT /v1/users/:user_id/invitations/:id/execute Take action on the invitation

Meetings

Resource Description
GET /v1/users/:user_id/meetings List all user meetings either as creator or as invitee
POST /v1/users/:user_id/meetings Create a meeting specifying a place, date, and time
GET /v1/users/:user_id/meetings/:id Get a meeting information
PUT /v1/users/:user_id/meetings/:id Update meeting information
POST /v1/users/:user_id/meetings/:id/invite Invite an user to a meeting
GET /v1/meetings/:id/:access_code Access a meeting

Users

Resource Description
GET /v1/users List users
POST /v1/users Add a new user
GET /v1/users/:id Get user information
PUT /v1/users/:id Update user information
