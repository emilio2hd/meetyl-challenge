Meetings / invite

POST /v1/users/:user_id/meetings/:id/invite
Invite an user to a meeting

How to Specify Invitation Recurrence¶ ↑

You can specify the recurrence through the { invitation: { recurrence: {} }}

Daily¶ ↑

Daily

recurrence: { type: 'daily' }

Every 3 days

recurrence: { type: 'daily', options: { interval: 3 } }

Weekly¶ ↑

Weekly

recurrence: { type: 'weekly' }

Every 2 weeks

recurrence: { type: 'weekly', options: { interval: 2 } }

Weekly on Mondays

recurrence: { type: 'weekly', options: { day: ['monday'] } }

Every 2 weeks on Mondays

recurrence: { type: 'weekly', options: { interval: 2, day: ['monday'] } }

Monthly¶ ↑

Monthly

recurrence: { type: 'monthly' }

Every 2 months

recurrence: { type: 'monthly', options: { interval: 2 } }

Monthly on the 15th day of the month

recurrence: { type: 'weekly', options: { day_of_month: [15] } }

Monthly on the 10th and 20th days of the month

recurrence: { type: 'weekly', options: { day_of_month: [10, 20] } }

Monthly on the 1st Tuesday and last Tuesday

recurrence: { type: 'weekly', options: { day_of_week: { tuesday: [1, -1] } } }

Every 2 months on the 1st Monday and last Tuesday

recurrence: { type: 'weekly', options: { interval: 2, day_of_week: { monday: [1], tuesday: [-1] } } }

Yearly¶ ↑

Yearly

recurrence: { type: 'yearly' }

Every 2 years

recurrence: { type: 'yearly', options: { interval: 2 } }

Yearly on the 100th days from the beginning and end of the year

recurrence: { type: 'yearly', options: { day_of_year: [100, -100] } }

Yearly in March

recurrence: { type: 'yearly', options: { month_of_year: [:march] } }

Yearly in January and February

recurrence: { type: 'yearly', options: { month_of_year: [:january, :february] } }

Examples

POST /v1/users/1/meetings/1/invite
{
  "invitation": {
    "invitee_id": 1,
    "recurrence": {
      "type": "weekly",
      "options": {
        "interval": 2,
        "day_of_week": {
          "monday": [
            1
          ],
          "tuesday": [
            -1
          ]
        }
      }
    }
  }
}
201
{
  "invitation": {
    "invitee": 1,
    "meeting_link": "http://test.com/v1/meetings/1/NCQ3JDJkMjUxNjhjLTMwOTQtNDc2YS05ZWIzLTIwODUxN2M5YTg3MA=="
  }
}
POST /v1/users/1/meetings/1/invite
{
  "invitation": {
    "recurrence": {
      "type": "badly"
    }
  }
}
400
{
  "errors": {
    "recurrence": [
      "The type 'badly' is not supported"
    ]
  }
}

Params

  • invitation : required [

    Must be a Hash

    ]

    Invitation info

    • invitee_id : required [

      Must be a Integer

      ]

      User invitee id

    • recurrence : optional [

      Must be a Hash

      ]

      Invitation recurrence

      • type : required [

        Must be a String

        ]

        Recurrence type

      • start_time : optional [

        Must be a DateTime

        ]

        When the recurrence should start

      • end_time : optional [

        Must be a DateTime

        ]

        When the recurrence should end

      • options : optional [

        Must be a Hash

        ]

        Recurrence options

        • interval : optional [

          Must be a Integer

          ]

          Recurrence interval

        • day : optional [

          Must be an array of String

          ]

          The weekday name(s). Used with type: weekly

        • day_of_month : optional [

          Must be an array of Integer

          ]

          The day(s) of the month. Used with type: monthly

        • day_of_week : optional [

          Must be a Hash

          ]

          The weekday name(s) and its position(s) on month (1st Tuesday, 2nd Monday). Used with type: monthly

        • day_of_year : optional [

          Must be an array of Integer

          ]

          The day(s) of the year. Used with type: yearly

        • month_of_year : optional [

          Must be an array of String

          ]

          The month(s) name. Used with type: yearly
